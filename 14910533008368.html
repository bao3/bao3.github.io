养成使用 Mac Brew 安装软件

2017/4/1

这是很奇怪的一种感觉，虽然我们可以在各官网下载软件，但是非常麻烦，于是就有了一个简单的工具，只要用命令行就可以完成需要的软件安装，当然是大多数软件，毕竟这也是需要志愿者来维护的

首先你要安装好 Brew

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

然后如我这样，安装所需要的软件，你可以使用 brew search 先查找软件。

brew install mac-vim git youtube-dl you-get &&\
brew cask install resilio-sync cleanmymac \
    dropbox aria2gui surge google-chrome \
    iterm2 bartender alfred vmware-fusion istat-menus \
    keka  ifunbox airflow fswatch veracrypt omnidisksweeper
