在 Vultr上安装 Arch linux

发表于 2015-08-16

下载 iso安装盘

Vultr 上支持自定义系统而不必限于已经存在的系统，所以你去customIOS 输入 iso的地址，让 Vultr给你下载好。我的 VPS是日本节点，所以使用了日本的地址:

http://ftp.jaist.ac.jp/pub/Linux/ArchLinux/iso/2015.08.01/archlinux-2015.08.01-dual.iso

创建系统

这个应该都会了吧， Deploy 一个新的，注意先择系统时，使用你刚才下载的 iso ，不过在创建好之后系统根本不会自动安装好 Arch Linux，只不过是用光盘启动了而已。

注意一点，如果你创建的时候很不幸选了SATA （更多存储空间）那就郁闷了，因为有可能会莫名各种小毛病，影响后面的安装。
我经历过，所以只建议你使用 SSD套餐
并且 SATA套餐不能制作 Snapshot而 SSD可以，基于此，相当于多了一个免费的备份镜像，数据是数据是无价的。

开始安装

我相信你应该是一个具有动手能力的人，因为接下来的安装需要使用很多命令行指令。请在 Web上选使用 VNC Console的方式来使用新创建的 VPS。

想写一个小说

缘起

这些年的发展，家乡也开始发生很大变化，在外工作的十多年使我觉得有必要把家乡的东西保存下来，于是便想到写小说。 将几个友人的成长编成小说，虽然在这微博横行，微信乱入的年代里，写这么长的文字似乎心力不足。但还是尽量吧。

书写环境

  • 电脑一台，2015 Rmbp 13” 原本是给老婆的礼物，不过她就是用来无广告看爱奇艺，太浪费了，于是被我霸占。
  • 使用了 Markdown 语言 ，尝试用这种新的语言尝试写作。
  • 使用 Macdown.app 作为写作前端，虽然自己依然是一个 VIM党人，不过不得不说在我没有适应 Markdown 这语言前还是要靠它来实时显示的。
  • 托管在 github 上
  • 使用 hexo 生成博客文章
  • 使用 syncthing 备份 hexo 到 VPS上，为的是防止意外

关于我自己

我是一个典型的 80后、85前，这个年龄的人经历了中国大部分的变革，一直是国家政策的实验对象，从出生、上学、高考、就业、买房、户籍甚至二胎。如今我所在的单位越来越多的是 90后新生代，我觉得应当写一下。

不过我只是一个 Geek，没有什么文笔，但愿能写出不差的故事来。
